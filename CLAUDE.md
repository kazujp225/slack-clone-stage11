# Slackクローン（ダミーデータ版）

## 目的
Slackの全画面をダミーデータで構築し、2カラムレイアウト + レスポンシブの基礎を習得する。

## 技術スタック
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- react-router-dom

## 画面構成
- サイドバー: 幅260px / 背景色 紫 #611f69 / チャンネル5つ
- ヘッダー: チャンネル名表示
- メッセージ一覧: ダミー10件
- 入力欄: 下部固定

## データ
全てダミーデータ。DB接続・認証はまだ行わない（Stage 14以降で対応予定）。

## レスポンシブ
375px幅でサイドバーをハンバーガー + Sheet に切り替える。
